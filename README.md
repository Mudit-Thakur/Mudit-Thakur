### 2️⃣ **Indian Beauty Market — End-to-End Retail Analytics Pipeline**

**Business Problem:**  
Retail business: 50,000 transactions (₹1.62 Cr revenue) but **no visibility** into SKU performance, customer health, or demand drivers.

**What I Built:**
- **ETL Pipeline:** Python (Polars) processing 50K transactions into clean data lake
- **Data Warehouse:** DuckDB with Parquet optimization for fast queries
- **BI Dashboards:** Power BI + DAX for SKU, customer, and revenue analysis
- **AI Layer:** LangChain + Groq (LLaMA 3.3) for natural-language querying

**Business Outcomes:**
✅ **Revenue concentration risk flagged:** Top 3 SKUs = 46% of revenue (hidden tail risk → inventory planning fixed)  
✅ **Customer segmentation:** Identified at-risk cohort (**302 customers = 30.2%** showing churn signals) → enabled proactive retention campaign  
✅ **Customer value stratification:** VIP + Loyal = **94% of total revenue** (strategic focus area)  
✅ **AI accessibility:** Natural-language agent lets business ask questions without SQL (proof of concept deployed)  

**Tech Stack:**  
`Python` `Polars` `DuckDB` `Parquet` `Power BI` `DAX` `LangChain` `Groq` `Git`
